# ShadowBlade · 中文术语表

权威翻译表。批量翻译时遵循这份表。**ShadowBlade 保留英文不译**（产品名/品牌名）。

## 产品语态

- 简洁、专业、不夸张。短句优先，14 字以内为佳。
- 不用感叹号。不用「赋能 / 抓手 / 闭环 / 链路」等行话。
- 数字保留阿拉伯数字，单位用中文（如 `28 秒`、`4 分钟`）。
- 英文产品名（Okta、Slack、Figma、TikTok 等）保留原文。
- 货币保留 `$`（美元），如 `$4,200/月`。
- 文件后缀、域名、变量名、代码保留英文。

- `cuts` → `条` 或 `个`（如 `387 条`）

- `renders / mo` → `次/月`

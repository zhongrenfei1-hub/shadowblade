文档

基于 ShadowBlade API 构建。

从简报渲染一条成片、实时查看流水线、把 MP4 投递到团队所有分发渠道。使用 Bearer Token 鉴权，生产环境可接入 SSO。

快速上手

从零到首个成片只需 7 分钟。

  • 安装 CLI 5 分钟
  • 从简报渲染 2 分钟
  • 拉取首个 MP4 1 分钟

# curl — 对照 Acme · Core 品牌套件渲染一条 30 秒主视觉
curl "https://api.shadowblade.io/v1/projects/101/render" \
  -H "Authorization: Bearer $SHADOWBLADE_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "template":   "hero-launch",
    "brand_kit":  "acme-core",
    "brief":      "春季发布 — 智能腕环...",
    "duration":   30,
    "aspect":     "9:16",
    "voice":      "alloy-en-female",
    "priority":   "normal",
    "webhook":    "https://acme.example.com/hooks/sb"
  }'

# → 202 Accepted
# { "run_id": 901, "eta_seconds": 272, "stage": "queued" }

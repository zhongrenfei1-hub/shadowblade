Render a cut in one POST.

Bearer token, the brief, the brand kit. The webhook lets you light up your distribution pipeline the moment the MP4 is ready.

Read the full reference Try in the workspace 

# curl — render a 30s launch hero against the Acme · Core brand kit
curl "https://api.shadowblade.io/v1/projects/101/render" \
  -H "Authorization: Bearer $SHADOWBLADE_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "template":   "hero-launch",
    "brand_kit":  "acme-core",
    "brief":      "Spring launch — wearable hub...",
    "duration":   30,
    "aspect":     "9:16",
    "voice":      "alloy-en-female",
    "priority":   "normal",
    "webhook":    "https://acme.example.com/hooks/sb"
  }'

# → 202 Accepted
# { "run_id": 901, "eta_seconds": 272, "stage": "queued" }
